This file is meant to explain each of the top level directories.  There should be a similar readme in the relevant folders.

eq_analysis   - contains the scripts used to generate the equilibration rmsf plots as well as the plot themselves.
fep_config    - contains everything necessary to run the dual topology fep
fep_data      - contains the final data and analysis script for the fep simulations
figures       - contains all plots used in the manuscript along with pymol structures that were used at various points in development, the force curves, and the movies
smd_config    - contains everything necessary to run our smd pipeline
smd_data      - contains all data and scripts that we ever used in the preparation of this manuscript
smd_rmsf_data - contains all data and scripts used to generate the rmsf comparison plot
